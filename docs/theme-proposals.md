# Theme and Mood Proposals

## Current theme categories

The app currently has these six theme categories:

1. Fantasy
2. Sci-Fi
3. Horror
4. StarWars
5. Harry Potter
6. Witcher

## Current moods

The app currently has these moods:

- Fun
- Serious
- Scary
- Silly
- Dramatic
- Chaotic
- Mysterious
- Adventurous

## Proposed bolder theme directions

These proposals are meant for **card lists and card items only**, while app chrome and navigation stay on the default theme.

### 1. Fantasy

- Palette: royal violet, antique gold, parchment cream, deep moss
- Feel: heroic, magical, storybook
- Card treatment: gold accent edge, parchment body, jewel-tone highlights

### 2. Sci-Fi

- Palette: electric cyan, ultraviolet, graphite, cold white
- Feel: high-tech, synthetic, neon
- Card treatment: sharp contrast, cool glow accents, dark steel surfaces

### 3. Horror

- Palette: dried-blood red, grave black, bruised plum, fog gray
- Feel: oppressive, eerie, dangerous
- Card treatment: dark cards with crimson accents and pale text contrast

### 4. StarWars

- Palette: desert sand, deep navy, lightsaber blue, rebel crimson
- Feel: cinematic, mythic, galactic adventure
- Card treatment: warm sand surfaces with bold heroic color hits

### 5. Harry Potter

- Palette: burgundy, antique brass, candlelight amber, ink navy
- Feel: academic magic, boarding-school mystery, cozy wonder
- Card treatment: parchment-like cards, brass accents, house-style richness

### 6. Witcher

- Palette: steel gray, oxblood, alchemy amber, winter blue
- Feel: gritty, weathered, monster-hunter realism
- Card treatment: scarred-metal accents, muted surfaces, ember highlights

## Sci-Fi vs StarWars distinction

These two should stop sharing the same visual family.

- **Sci-Fi** should feel cold, precise, technological, neon, and interface-driven.
- **StarWars** should feel warm, mythic, cinematic, sandy, and heroic.

A good rule:

- Sci-Fi = cyan / violet / graphite
- StarWars = sand / blue / crimson / navy
